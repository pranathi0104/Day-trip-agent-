# AI Travel Multi-Agent System

An AI-powered travel planning assistant built using Google ADK and Gemini.

This project uses multiple AI agents that collaborate to help users plan trips, check weather, discover food places, and generate travel itineraries.

---

## Features

- 🌤️ Weather Agent
  - Provides weather forecasts and travel climate suggestions

- 🍜 Food Recommendation Agent
  - Suggests local foods and restaurants

- ✈️ Travel Planner Agent
  - Creates travel itineraries and trip plans

- 🤖 Multi-Agent Collaboration
  - Agents work together to generate complete travel assistance

- 🧠 Powered by Gemini + Google ADK

---

## Tech Stack

- Python
- Google ADK
- Gemini API
- Google Colab
- Streamlit (UI)
- FastAPI (Backend)
